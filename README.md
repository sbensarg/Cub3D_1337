## Usage

```bash
make
./cub3D ../map.cub
```
or
```bash
make
./cub3D ../map.cub --save
```
for save the first rendered image in bmp format.

## Controls

You can move around the maze with ```W``` ```A``` ```S``` ```D```.

You can also rotate the view with the directional arrows ```←``` and ```→```.

To exit the game, press ```ESC``` or click the red cross.
